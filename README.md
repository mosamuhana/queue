# @devteks/queue

Fast Queue implementation in javascript

## how to use
`npm install @devteks/queue --save` 

# Usage:
import:
```javascript
const Queue = require('@devteks/queue');
```
OR
```typescript
import Queue from '@devteks/queue';
```

# API
-------------------
| Method | Description |
|--------|-------------|
| `enqueue` | add item to the end of the queue |
| `dequeue` | remove item from the front of the queue |
| `clear` | clear queue |
| `get size` | get current size  |
